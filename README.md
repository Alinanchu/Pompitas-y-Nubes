# Proyecto cuisi

## Para la base inicialmente
```bash
cd backend
npm install express mongoose cors
```

## Para insertar datos
```bash
cd backend
node insertarProductos.js
node insertarcategorias.js
node insertarcolores.js 
node insertartallas.js
node insertartestimoniales.js
```

## Para levantar la aplicacion 
## Instrucciones
1. Abrir Visual Studio Code.
2. En una terminal, ir a `backend`:
```bash
cd backend
npm install
npm start
```
3. En otra terminal, ir a `frontend`:
```bash
cd frontend
npm install
npm run dev
```
4. Abrir `http://localhost:5173` en el navegador.
